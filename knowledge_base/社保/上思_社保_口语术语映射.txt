业务域：社保办理。

交农保 → 城乡居民养老保险 → 关联意图 I001、I002。
社保卡 → 社会保障卡 → I009。
办医保 → 基本医疗保险参保或医保待遇 → I008（医保窗口）。
养老金 → 基本养老保险待遇（城乡或职工）→ I003、I011。
五险一金 → 职工社会保险与住房公积金等 → I004、I006 等。
续保（灵活就业语境）→ 灵活就业人员续缴职工养老保险 → I007。
停保 → 暂停缴费或单位减员 → 需进一步询问职工或城乡、是否离桂等。
退休金 → 养老金 → I003、I011。
打工交的社保 → 职工基本养老保险 → I005、I004、I006。
老家交的养老 → 城乡居民养老保险 → I001 至 I003。
